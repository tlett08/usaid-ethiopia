# usaid-ethiopia
Essay and underlying project submitted as part of my application to the Master of Applied Data Science program at University of Michigan (currently enrolled). Analyzes relationship between USAID spending and child mortality in Ethiopia. Data processing tasks, EDA and regression model are contained in "USAID Spending Ethiopia_GitHub.ipynb." Please see "Ethiopia Essay_Final.pdf" for the essay itself. Remaining files are CSVs containing the underlying data collected from State Department and UNICEF. Data files from WHO and World Bank not included as the features drawn from this file were dropped from the final model.
